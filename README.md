# dict
Command Line Dictionary Tool that uses Wordnik API

## Setup
1. Install node, npm.
2. Download the repository, unzip the repository.
    npm link
3. The possible commands are,
    dict def <word>
    dict syn <word>
    dict ant <word>
    dict ex <word>
    dict dict <word>
    dict <word>
    dict play
    dict
    dict help
